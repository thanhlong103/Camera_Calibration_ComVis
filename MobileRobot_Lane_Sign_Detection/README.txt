## Run file MAIN.py to initiate the main loop for the robot to detect lanes and traffic signs

- To find the shape of the live cam/ video: get_videoShape.py

- To find the desired regions of interest: select_ROI.py

- To find the HSV of the signs' colors: get_HSV.py

- To test the working of lane detection and sign detection without the Raspberry and Arduino: test_main.py

- To find all helper functions used in MAIN.py: utlis.py
